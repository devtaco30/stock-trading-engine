# 📚 NPL 투자 개념 정리

## 1. NPL이란 무엇인가?

### 1-1. NPL의 정의
**NPL (Non-Performing Loan) = 부실채권**
- 은행에서 부동산 담보 대출을 받았는데, **이자를 3개월 이상 연체**한 채권
- 예시: 
  - 김철수가 은행에서 10억 빌려서 강남 아파트 샀음
  - 매달 이자 500만 원 내야 하는데 3개월째 안 냄
  - 은행: "이거 부실채권이네" → NPL로 분류

### 1-2. NPL은 어떻게 만들어지나?

**일반적인 흐름:**
```
1. 김철수, 은행에서 10억 대출 (강남 아파트 담보)
   → 은행이 아파트에 "근저당권" 설정 (담보 잡음)

2. 김철수, 사업 망해서 이자 못 냄 (3개월 연체)
   → 은행 장부에 "부실채권(NPL)" 기록

3. 은행, 손실 줄이려고 NPL을 자산관리회사에 매각
   → "10억짜리 채권인데 7억에 팔게" (30% 할인)

4. 자산관리회사, 이걸 다시 잘게 쪼개서 개인 투자자들에게 판매
   → 또는 소형 투자사, 플랫폼에게 재판매
```

### 1-3. NPL의 종류

**담보부 NPL (일반적)**
- 부동산, 공장 설비, 차량 등 담보가 있는 채권
- 위험도: 중 (담보 가치만큼은 회수 가능)
- 수익률: 10~30%
- 예시: 강남 아파트 담보 대출 채권

**무담보 NPL**
- 신용카드, 신용대출 등 담보 없는 채권
- 위험도: 높음 (못 받을 수도 있음)
- 수익률: 30~100% (또는 0%)
- 예시: 신용카드 연체 채권

## 2. NPL 투자로 돈 버는 방법

### 2-1. 방법 1: 경매 낙찰 후 배당 받기
**개념:**
- 채권을 갖고 있으면 경매 시 우선권이 있음
- 경매로 부동산이 팔리면 채권자에게 먼저 배당

**예시:**
```
1. NPL 채권 매입: 7억 (원금 10억의 70%)
2. 경매 진행: 12억에 낙찰
3. 배당 받기: 7억(원금) + 연체이자(약 1억) = 8억
4. 수익: 1억 (14%)
```

**장점:**
- 직접 낙찰받을 필요 없음
- 배당만 받으면 됨
- 비교적 안전함

### 2-2. 방법 2: 직접 낙찰받아서 되팔기
**개념:**
- 채권자는 경매에서 유리한 지위
- 낙찰 후 시세차익으로 재판매

**예시:**
```
1. NPL 채권 매입: 7억
2. 경매 참여: 채권자 지위로 10억에 낙찰
3. 실제 납부: 10억 - 7억(채권액) = 3억만 납부
4. 부동산 매각: 15억에 판매
5. 수익: 15억 - 10억 = 5억 (71% 수익)
```

**장점:**
- 높은 수익 가능
- 채권자 우선권 활용

**단점:**
- 직접 관리 필요
- 추가 자금 필요 (잔금)

### 2-3. 방법 3: 채무자와 협상
**개념:**
- 채무자와 직접 협상하여 상환 받기

**예시:**
```
1. NPL 채권 매입: 7억 (원금 10억)
2. 채무자와 협상: "이자 다 깎아줄게, 원금 8억만 갚아"
3. 채무자 상환: 8억
4. 수익: 1억 (14%)
```

**장점:**
- 경매 절차 불필요
- 빠른 회수

**단점:**
- 협상 실패 가능
- 채무자 지급 능력 필요

## 3. NPL 투자의 장점과 단점

### 3-1. 장점

**높은 수익률:**
- 담보부 NPL: 10~30%
- 위험 감수 시: 30~100%

**안정성 (담보부 기준):**
- 1순위 근저당권은 회수 가능성 높음
- 담보 가치가 바닥 역할

**세금 혜택:**
- 배당 소득은 비과세 (채권자 지위)
- 일반 부동산 매매보다 세금 부담 적음

**정보 우위:**
- 채권자는 담보물 정보 접근 용이
- 경매 전 물건 상태 파악 가능

**경매 우선권:**
- 채권자는 경매에서 유리
- 낙찰 실패해도 배당으로 회수

### 3-2. 단점 (리스크)

**권리 분석의 어려움:**
- 선순위 채권, 임차인 권리 등 복잡
- 법적 지식 필요
- 초보자 접근 어려움

**부동산 시장 의존:**
- 부동산 가격 하락 시 손실
- 감정가 예측 실패 시 위험
- 경기 변동에 민감

**장기간 소요:**
- 경매 진행: 6개월~1년
- 채무자 협상: 예측 불가
- 자금 회전 느림

**유찰 위험:**
- 경매 유찰 시 원금 손실 가능
- 담보 가치 하락 시 문제
- 재경매 시 더 낮은 가격

**제한적 물건:**
- 좋은 물건은 기관이 선점
- 개인에게 돌아오는 건 한정적
- 정보 비대칭

**협상 어려움:**
- 채무자와 합의 어려움
- 법적 분쟁 가능성
- 시간 소모

## 4. NPL 투자 절차

### 4-1. 전통적인 절차 (기관 투자자)

```
1. NPL 물건 발굴
   - 자산관리회사(AMC)에서 매물 확인
   - 은행, F&I 자회사 등에서 구매

2. 실사 (Due Diligence)
   - 등기부등본 확인 (선순위 채권 여부)
   - 현장 방문 (담보물 상태)
   - 임차인 확인
   - 감정가 재평가

3. NPL 매입
   - 가격 협상
   - 계약 체결
   - 채권 양도 절차

4. 회수 전략 수립
   - 경매 참여? vs 배당만?
   - 채무자 협상?
   - 직접 매각?

5. 실행
   - 경매 입찰
   - 또는 채무자 협상
   - 잔금 납부 (낙찰 시)

6. 정산
   - 배당금 수령
   - 또는 부동산 매각
   - 수익 실현
```

### 4-2. 개인 투자자 절차 (플랫폼 이용)

```
1. 플랫폼 가입
   - 회원 가입
   - 본인 인증 (KYC)
   - 예치금 입금

2. 상품 선택
   - 플랫폼에 올라온 NPL 상품 확인
   - 수익률, 위험도, 기간 검토
   - 상세 정보 (등기부, 사진 등) 확인

3. 투자 신청
   - 투자 금액 입력
   - 선착순 또는 청약 방식
   - 투자 확정

4. 대기
   - 플랫폼이 경매 진행
   - 진행 상황 알림 수신
   - 3~12개월 대기

5. 수익 정산
   - 플랫폼이 배당금 또는 매각대금 정산
   - 투자자 계좌로 입금
   - 수익 확인
```

## 5. NPL 시장 구조

### 5-1. NPL 유통 경로

```
[1단계] 은행 → 대형 자산관리회사 (AMC)
- 대표: 한국자산관리공사(캠코), 우리F&I, 하나F&I 등
- 규모: 수백억~수천억 단위
- 할인율: 50~70% (원금 대비)
- 예: 1,000억 채권을 500억에 매각

[2단계] 대형 AMC → 중소형 자산관리회사
- 규모: 수십억 단위로 쪼갬
- 할인율: 60~80%
- 재판매 또는 자체 관리

[3단계] 중소형 회사 → P2P 플랫폼 또는 투자조합
- 규모: 수억~수십억
- 개인 투자자 참여 가능
- 플랫폼이 조각으로 쪼갬

[4단계] 플랫폼 → 개인 투자자
- 규모: 10만 원부터 가능
- 조각투자 형태
- 소액 분산 투자
```

### 5-2. 주요 플레이어

**자산관리회사 (AMC):**
- 한국자산관리공사 (캠코): 공기업
- 우리F&I, 하나F&I, 키움F&I: 금융그룹 자회사
- 중소형 자산관리회사 다수

**P2P 플랫폼:**
- 테라펀딩, 어니스트펀드, 펀다 등
- 소액 투자자와 NPL 연결
- 온투법 규제 받음

**대부업체:**
- NPL 매입 후 채무자에게 직접 추심
- 무담보 NPL 주로 취급

## 6. 개인이 NPL에 투자하는 방법

### 6-1. 과거: 개인 직접 매입 (2016년 이전)
- 개인이 자산관리회사에서 NPL을 직접 살 수 있었음
- 지금은 **법으로 금지됨** (2016년 7월부터)
- 이유: 불법 추심 등 사회적 문제

### 6-2. 현재: 간접 투자만 가능 (2016년 이후)

**방법 1: P2P 플랫폼**
- 플랫폼이 NPL 사서 쪼갬
- 개인은 소액(10만 원~)으로 투자
- 온투법 규제 (투자 한도 제한)
- 예: 테라펀딩, 어니스트펀드 등

**방법 2: 펀드 또는 투자조합**
- 여러 명이 모여서 조합 만듦
- 조합이 NPL 매입
- 수익 나누기
- 사모: 49인 이하, 공모: 무제한

**방법 3: 등록된 법인 통해서**
- 대부업 등록한 회사가 NPL 매입
- 개인은 그 회사 지분이나 채권에 투자

### 6-3. 용어 설명: GP와 LP

**투자조합의 구조:**
```
투자조합 = GP(관리자) + LP(투자자들)

GP (General Partner) = 업무집행조합원
- 역할: 조합 운영, NPL 선택, 경매 참여, 관리
- 책임: 무한책임 (빚 생기면 개인 재산으로 갚아야 함)
- 보상: 운용 수수료 + 성과보수
- 예: 플랫폼 운영사

LP (Limited Partner) = 유한책임조합원
- 역할: 돈만 냄 (투자만 함)
- 책임: 유한책임 (넣은 돈만큼만 손해)
- 보상: 투자 수익
- 예: 일반 개인 투자자들
```

**예시:**
- 우리 플랫폼(GP)이 NPL 투자조합 만듦
- 개인 49명(LP)이 각자 1,000만 원씩 투자
- 총 4억 9천만 원으로 NPL 매입
- GP는 관리하고, LP는 수익 배당 받음

## 7. 최근 NPL 시장 동향 (2024~2026)

### 7-1. NPL 매물 급증 원인

**원인 1: 코로나 특수 종료**
- 2020~2022: 코로나 금융 지원으로 연체율 낮음
- 2023~: 지원 종료, 실물 경기 악화
- 결과: 은행권 NPL 급증

**원인 2: 고금리 장기화**
- 기준금리 3.5% 유지 (2023~2025)
- 이자 부담 증가 → 연체 증가
- 특히 부동산 PF 대출 부실

**원인 3: 부동산 경기 침체**
- 부동산 가격 하락
- 담보 가치 하락
- LTV 초과 → 은행 위험 증가

**원인 4: 저축은행 건전성 악화**
- 저축은행 연체율: 6.55% (2023) → 8.80% (2024 1Q)
- 부동산 PF, 개인 사업자 대출 부실
- NPL 매각 압박

### 7-2. 저축은행 공동 매각

**배경:**
- 저축은행 NPL은 규모가 작음 (100억~200억)
- F&I 회사들이 관심 없음 (은행 NPL은 1,000억 이상)

**해결책:**
- 18개 저축은행 모여서 1,360억 규모로 **공동 매각** (2025)
- 규모 키워서 F&I 관심 유도
- 우리F&I, 키움F&I, 하나F&I 입찰 참여

**전망:**
- 공동 매각 정례화 예상
- 카드사, 캐피탈도 공동 매각 검토 중
- NPL 시장 확대

### 7-3. 2026년 시장 전망

**공급 측면:**
- NPL 매물 계속 증가
- 은행, 저축은행, 카드사 모두 부실 증가
- 2금융권 NPL "큰 장" 예상

**수요 측면:**
- F&I 회사들 매입 확대
- P2P 플랫폼, 투자조합 증가
- 개인 투자자 관심 증가

**투자 환경:**
- 경쟁 치열 (수익률 하락 가능)
- 플랫폼 난립 (선택 중요)
- 규제 강화 가능성

## 8. NPL 투자 시 주의사항

### 8-1. 실사의 중요성
- 등기부등본 꼼꼼히 확인
- 선순위 채권 여부
- 임차인 권리 (전세권, 임차권 등)
- 현장 방문 필수

### 8-2. 플랫폼 선택
- 금융 라이선스 확인 (온투법 등록 여부)
- 과거 실적 확인
- 수수료 구조
- 투자자 보호 장치

### 8-3. 분산 투자
- 한 상품에 몰빵 금지
- 5~10개 이상 분산
- 담보 종류도 분산 (부동산, 동산 등)

### 8-4. 투자 기간
- 단기 회수 어려움 (최소 6개월)
- 유동성 부족
- 급전 필요 시 곤란

### 8-5. 세금
- 배당 소득 비과세 (채권자 지위)
- 부동산 직접 취득 시 취득세, 양도세
- 플랫폼 투자 시 세금 구조 확인
