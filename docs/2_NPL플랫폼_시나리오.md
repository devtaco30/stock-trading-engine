# 📱 NPL 공동투자 플랫폼 - 프로젝트 시나리오

## 1. 서비스 개요

### 1-1. 플랫폼 컨셉
- **이름**: (미정)
- **컨셉**: 개인 투자자들이 NPL에 소액으로 공동 투자하여 수익을 추구하는 플랫폼
- **차별점**: 기관만 참여하던 NPL 시장에 개인들이 소액으로 진입

### 1-2. 플랫폼 역할
- NPL 발굴 및 실사 (물건 선정)
- 투자자 모집 (앱)
- 경매 참여 및 관리 (GP 역할)
- 수익 정산 및 배당

### 1-3. 수익 구조

**투자자 입장:**
```
1. 앱에서 마음에 드는 NPL 선택
2. 투자금 입금 (예: 1,000만 원)
3. 경매 낙찰되면 지분 확정
4. 배당 받기 (6개월~1년 후)
```

**플랫폼 입장:**
```
1. NPL 발굴 및 매입
2. 투자자 모집 (앱)
3. 수수료 수입:
   - 투자금의 1~2% (운용 수수료)
   - 수익의 10~20% (성과보수)
```

## 2. 법적 구조 (2026년 기준)

### 2-1. 사모 투자 모델 (초기 단계)

**인원 제한:**
- 최대 49인 (자본시장법 제9조 제19항)
- 50인 이상 시 증권신고서 제출 필요 (공모 전환)
- 최소 2인 (GP 1인 + LP 1인)

**특징:**
- 폐쇄형 클럽 딜 형태
- 공개 모집 불가, 지인 네트워크 통한 모집만 가능
- 진입 장벽 낮음, 규제 간소함

**투자 금액:**
- 1인당 최소 투자금: 100만 원 이상 (벤처투자법 시행령)
- 조합 총 출자금: 1억 원 이상

### 2-2. 공모 투자 모델 (확장 단계)

**인원 제한:**
- 무제한 (증권신고서 수리 후)

**소액공모 특례:**
- 조달 금액 10억~30억 미만 시 간소화 공시
- 1인당 연간 투자 한도: 1,000만~2,000만 원

**특징:**
- 조각투자(STO) 방식
- 최소 투자금: 5,000원~10,000원 단위

### 2-3. 플랫폼 운영 요건

**필수 법인 등록:**
- 대부채권매입추심업: 자본금 5억 원 (NPL 매입용)
- 온라인소액투자중개업: 자본금 5억 원 (플랫폼 운영)

**발행/유통 분리 원칙:**
- 발행사: NPL 소유 법인 (SPC 또는 신탁)
- 유통사: 플랫폼 법인 (거래 중개)

### 3-2. 상품 예시: "강남 아파트 NPL"

**물건 정보:**
- 위치: 서울 강남구 대치동
- 담보: 아파트 35평

- 목표 모집액: 13억 원
- 모집 정원: 49명
- 1인당 투자 금액: 2,650만 원 (균등 배분)
- 예상 수익률: 15~20%
- 예상 기간: 6개월

### 3-3. 유저 시나리오 1: 이현우 (성공 케이스)

**프로필:**
- 나이: 37세
- 직업: IT 회사 팀장
- 투자 경험: 주식, 코인 경험 있음
- 가용 자금: 5,000만 원
- 투자 성향: 중위험 중수익

**Phase 1: 상품 발견 (D-3)**
- 앱 알림: "새로운 NPL 상품 오픈 예정"
- 상품 상세 페이지 확인
- 등기부등본, 사진, 주변 시세 확인
- 예상 수익률 15% 확인
- 투자 결정

**Phase 2: 상품 오픈 전 (12:00 전)**
- 11:55 - 앱 접속

- 11:59 - 투자 대기 화면 진입
- 11:59:50 - 카운트다운 시작

**Phase 3: 선착순 경쟁 (12:00:00)**
- 12:00:00 - 정확히 12시에 "투자하기" 버튼 클릭
- 동시 접속자: 약 10만 명이 같은 상품에 접근
- 경쟁률: 10만 명 중 49명만 투자 가능
- 12:00:01 - 화면에 "투자 신청 완료" 메시지 표시
- 이현우는 47번째 투자자로 등록됨
- 푸시 알림: "30분 내 입금하세요"

**Phase 4: 입금 및 확정**
- 12:05 - 계좌로 입금 안내 문자 수신
- 12:30 - 2,650만 원 입금 완료
- 13:00 - "투자 확정" 상태로 전환
- 지분: 13억 중 2,650만 원 (약 2.04%)
- 앱에 "내 투자" 탭에 표시됨

**Phase 5: 경매 진행 (2주 후)**
- D+14 - 플랫폼이 49명의 자금(13억)으로 경매 입찰
- 경매 당일 실시간 알림
- 낙찰 성공: 12억 5천만 원에 채권 취득
- 이현우 보유 자산: "강남 아파트 NPL 지분 2.04%" (수익증권 형태)
- 앱에서 진행 상황 확인 가능

**Phase 6: 수익 정산 (6개월 후)**
- D+180 - 채무자가 원금 + 이자 상환: 15억 원
- 플랫폼 수수료(2%): 3,000만 원
- 순수익: 14억 7,000만 원
- 이현우 배당: 14억 7,000만 원 × 2.04% = 2,998만 원
- 실제 수령액: 2,998만 원 (입금)
- 수익률: (2,998만 - 2,650만) / 2,650만 = 13.1%
- 투자 기간: 6개월 → 연환산 26.2%

### 4-3. 유저 시나리오: 김소액 (소액 투자자)

**프로필:**
- 나이: 25세
- 직업: 대기업 신입사원
- 투자 경험: 거의 없음

**Phase 2: 모집 완료 (1주 후)**

- 김소액은 소액 투자자 그룹 (하위 30%)
- 모집 완료: 55억 달성

**Phase 3: 경매 및 낙찰**
- 플랫폼이 55억으로 경매 참여
- 낙찰 성공: 50억

**Phase 4: 정산 수익 (1년 후)**

- 수익률: (11만 4,660원 - 10만 원) / 10만 원 = 14.7%
- "10만 원으로 1만 4천 원 벌었네, 괜찮은데?"
