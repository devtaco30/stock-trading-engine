# 주식 거래 및 자산 코어 엔진

> Java/Spring Boot 기반 증권 계좌 시스템 백엔드

## 프로젝트 개요

주식 거래 시 발생하는 T+2 결제, 증거금, 미수금, 강제 청산 로직을 구현하는 백엔드 엔진입니다.

증권사 계좌 시스템의 핵심 비즈니스 로직을 Java로 정확하게 구현하는 게 목표입니다.

## 핵심 시나리오

### 1. 기본 주문 및 체결
실시간 증거금 체크와 체결 처리

### 2. T+2 시차 활용
매도 예정금액을 담보로 즉시 재매수 (회전매매)

### 3. 미수금 발생 및 연체
결제일 배치로 미수 계좌 상태 전환

### 4. 강제 청산
미수 미납 시 시스템이 자동으로 보유 주식 매도
